프로세스별 공통 프롬프트 가이드

본 가이드는 실무자가 5시간의 필수 훈련 과정 동안 생성형 AI와 소통하며
자생적 AX 역량을 내재화할 수 있도록 설계된 표준 명령어 템플릿 세트입니다.

프롬프트 다운로드
AI 코치(컨설턴트) 안내 지침

본 문서는 경남경영자총협회 중소기업AI훈련확산센터의 '생성형 AI 기반 업무관리 웹보드 개발 및 리포트 자동화 훈련' 지침에 의거하여 작성되었습니다. 코치진은 본 가이드를 활용하여 현장 실무자가 AI와 대화하며 주도적으로 코드를 제어할 수 있도록 매칭 지도해 주시기 바랍니다.

■ 1단계: 직무 이해 및 AI 기획
필수 1시간

훈련 목적

AI에게 실무자의 도메인(직무) 지식을 학습시키고, 시스템에 담을 핵심 흐름을 구조화합니다.

"나는 제조업에서 [본인의 직무: 예-품질관리/자재구매/설비보전 등]을 담당하고 있는 실무자야. 
우리 회사는 중소기업이라서 전용 시스템이 없고 주로 엑셀로 업무를 처리하고 있어. 
1. 이 직무를 수행할 때 가장 핵심적인 업무 프로세스 5가지 흐름을 정의해 줘.
2. 이 중에서 매일 데이터를 입력하고 일일/주간 보고서를 작성할 때 시간이 가장 많이 걸리는 '병목 구간'이 어디일지 실무 관점에서 분석해 줘.
3. 이 병목 업무를 자동화하기 위한 No-Code 웹 프로그램(웹보드)을 기획하려고 하는데, 어떤 기능과 데이터 항목(필드)들이 필수로 들어가야 할지 제안해 줘."
■ 2단계: 프로그램 생성
필수 1시간

훈련 목적

1단계에서 도출된 기능을 바탕으로, 단일 파일로 동작하는 웹 프로그램의 기초 뼈대 코드를 구축합니다.

"[역할] 너는 제조업 [직무명] 업무 효율화 프로그램 개발 전문가야.
[환경] 외부 서버나 DB 연동 없이 브라우저에서 오프라인으로 즉시 실행 가능한 단일 'index.html' 파일 코드를 작성해 줘. Bootstrap 5와 Chart.js 라이브러리를 CDN으로 포함하고, 데이터는 브라우저 LocalStorage에 JSON 형태로 저장·확인되도록 해줘.
[기능] 
1. 핵심 데이터 입력 폼 (데이터 항목: [1단계에서 제안된 항목들]) 및 실시간 누적 현황판(대시보드)
2. 주요 현황을 시각적으로 보여주는 그래프 1종 (Chart.js 활용)
[디자인] 현장 실무자가 쓰기 편하도록 가독성이 높은 모던한 [원하는 테마: 예-다크모드 또는 블루 테마] UI.
[제약조건] 코드 생략 없이 메모장에 복사해서 바로 'index.html'로 저장해 실행할 수 있는 전체 완성 코드로 주석을 포함해서 작성해 줘."
■ 3단계: 데이터 검증 및 연동
필수 2시간

훈련 목적

생성된 웹 프로그램에 기업의 실제 데이터를 적용하고, 발생하는 오류를 자기주도적으로 수정합니다.

"위에서 네가 만들어준 'index.html' 코드를 실행해서 우리 회사의 실제 샘플 데이터를 입력해 봤어. 그런데 다음과 같은 문제와 수정 요청 사항이 발생했어. 기존 코드에서 이 부분을 수정한 전체 코드를 다시 작성해 줘.
1. [오류 및 수정사항 기술: 예 - '데이터를 입력하고 추가 버튼을 눌렀는데 테이블에 실시간으로 반영이 안 돼', '기존에 축적된 엑셀 데이터를 복사해서 한 번에 붙여넣을 수 있는 엑셀 업로드(SheetJS) 기능이 필요해']
2. 실무에서 데이터 입력 실수를 방지할 수 있도록, [특정 조건: 예 - 재고가 10개 이하로 떨어지거나, 불량률이 5%를 초과할 때] 화면에 직관적인 경고 알림(Highlight 및 텍스트 문구)이 뜨는 시스템 안정화 로직을 추가해 줘."
■ 4단계: 리포트 고도화 및 UI 최적화
필수 1시간

훈련 목적

경영진 보고 업무를 혁신할 수 있도록 '원클릭 PDF 리포트 다운로드' 기능을 강화하고 화면을 최적화합니다.

"이제 프로그램의 기본 기능은 잘 작동해. 마지막으로 이 웹보드를 관리자나 경영진 보고용 시스템으로 쓸 수 있게 보고 기능을 완성하고 싶어.
1. 화면에 축적된 대시보드 현황과 핵심 통계 데이터를 깔끔한 표준 비즈니스 보고서 양식으로 레이아웃을 잡아서, 원클릭으로 '일일 업무 보고서(PDF)'를 다운로드할 수 있는 기능을 내장해 줘 (html2pdf 또는 jsPDF 라이브러리 CDN 활용).
2. 대시보드의 주요 지표 박스들과 그래프가 한눈에 들어오도록 UI 디자인을 더 세련되게 리디자인해 주고, 마우스 오버(Hover) 효과 등 시각적 완성도를 높여줘.
3. 전체 완성된 'index.html' 최종 코드를 전달해 줘."
■ [심화/선택] 5단계: 현장 배포 및 확산
심화 2시간

훈련 목적

완성된 파일의 상시 구동 인프라를 구축하고 타 부서로의 AI 활용 확산을 유도합니다.

"완성된 'index.html' 파일을 우리 부서원들과 공장 현장 작업자들도 태블릿이나 모바일로 접속해서 함께 사용할 수 있게 하고 싶어. 
1. 외부 서버 비용 없이 GitHub Pages를 활용해서 이 파일을 무료 웹 주소(URL)로 배포하고 상시 구동하는 방법을 초보자 눈높이에 맞춰 단계별 가이드로 설명해 줘.
2. 다른 부서에서도 이 AI 기반 업무 자동화 방식을 벤치마킹할 수 있도록, 내가 이 프로그램을 기획하고 AI와 소통하며 개발한 프로세스를 '사내 AI 표준 활용 가이드북' 양식(텍스트 초안)으로 정리해 줘."
